# 流程

# 1. 服务器开启 SSH 服务

首先需要在服务器上开启 SSH 服务，参考：

https://blog.csdn.net/qq_45164331/article/details/122533327

# 2.安装 remote SSH

vscode 在扩展中查找 Remote - SSH 并安装：

# 3.打开配置文件

打开 .ssh/config 配置文件

# 4.配置文件设置

根据实际配置：

Host 名称            # 自己随便取的
    HostName ip     # 服务器ip
    Port 22         # 通过ssh连接服务器，服务器必须开启ssh
    User   name     #  linux用户名称
1
2
3
4

# 5.打开连接

选中服务器后右键，选择在当前窗口或者新窗口打开服务器文件（需要服务器密码）

# 6.指定打开目录

新建终端，进入想要打开的项目后，使用命令code在当前项目下打开 vscode，之后如果想要打开同一个项目，可以直接在 SSH targets 下找到项目然后右键连接
